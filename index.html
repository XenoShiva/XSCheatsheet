6. Here we will provide a table with various html tags and their uses.

'
HTML TAGS
TAG USES
<!DOCTYPE html> An instruction declaration. It tells the browser what type of document to expect, along with what version of HTML is being used in the document.
<head> the metadata for a web page. Metadata is information about the page that isn’t displayed directly on the web page. Usually contains links to css stylesheet, and font links.
<title> Allows you to dictate a title in the tab of the website. Insert in head.
<h1-h6> h1 used for main headings. All other smaller headings are used for subheadings.
<div> Short for “division” or a container that divides the page into sections. These sections are very useful for grouping elements in your HTML together.
<p> The "paragraph" tag. Used to dictate lines of text.
<span> Short pieces of text or other html. used to seperate small pieces of content that are on the same line as other content.
<em> Emphasizes text in itallics.
<strong> Emphasizes text in bold.
<br> Creates a line break between elements.
<ul> Unordered list element. will create a list prompt with bullet points.
<ol> Order list element. Will create a list prompt in a numbered order.
<li> Defines the information contained in a list
<img src="image address link"/> Allows to insert images to the webpage. Must provide the address link in the defined area. Self closing tag.
<img src="image address link" alt="description"/> "alt" attribute can add description to your images to aid in diversity and vision impaired. Also utilized in SEO.
<video src="video link" controls> Allows insertion of videos. Requires closing tag. text between brackets can define text that is displayed if connection can not be made. Usually filled with "Video Not Supported
<a> Anchor element. Allows you to utilize links to other webpages, or specific parts of the current page
<table> Defines the skeleton of a data table
<th> Defines the title of the table. Using the "col" attribute we can define the top columns Using the "row" attribute we can define the left side rows.
<tr> Defines the table's rows. Where data can then be inserted.
<td> Defines the row's data. (each section is defined horizonally, requiring a new element for each block)
<thead> Sections off the head of the table in the element family, Making things easier to differentiate.
<tbody> Sections off the body of the table.
<tfoot> Sections off the end/bottom of the table. Usually to define the sums of the tables data.

This is not an account of all html tages, or attributes, but a brief summary. Reference W3 Schools to see a full list.
